# 튜브레이더 유튜브 홍보영상 대본

> 예상 길이: 3분 30초 ~ 4분
> 형식: 화면 녹화 + 자막 + 나레이션
> BGM: 밝고 경쾌한 테크/스타트업 느낌

---

## [INTRO] 0:00 ~ 0:15 — 후킹

**화면:** 유튜브에서 뭘 올릴지 고민하는 모습 / 경쟁 채널 일일이 검색하는 모습

**나레이션:**
"매일 유튜브 트렌드 확인하고, 경쟁 채널 하나하나 들어가서 분석하고...
소재 찾느라 영상 만들 시간이 없진 않으셨나요?"

**자막:** 소재 찾기에 하루를 다 쓰고 계신가요?

---

## [문제 제기] 0:15 ~ 0:30

**화면:** 유튜브 검색 → 탭 여러 개 열린 모습 → 엑셀에 정리하는 모습

**나레이션:**
"채널 분석은 수작업, 트렌드 파악은 감에 의존, 소재 발굴은 끝없는 검색의 반복.
크리에이터에게 필요한 건 데이터 기반의 똑똑한 도구입니다."

**자막:** 감이 아닌, 데이터로 승부하세요

---

## [솔루션 소개] 0:30 ~ 0:50

**화면:** 튜브레이더 메인 화면 등장 (로고 + 대시보드)

**나레이션:**
"튜브레이더를 소개합니다.
유튜브 채널 분석부터 소재 발굴, AI 바이럴 분석까지.
크리에이터를 위한 올인원 유튜브 데이터 플랫폼입니다."

**자막:** 튜브레이더 — 크리에이터를 위한 올인원 유튜브 분석 플랫폼

---

## [핵심 기능 1] 0:50 ~ 1:15 — 채널 모니터링

**화면:** 내 모니터링 리스트 화면 시연 — 채널 추가, 그룹 생성, 영상 카드 목록

**나레이션:**
"관심 채널을 그룹별로 저장하고, 최신 영상의 조회수와 바이럴 점수를 한눈에 확인하세요.
채널 비교 분석으로 경쟁 채널과의 차이를 차트로 비교할 수 있고,
급등 레이더가 급성장 중인 채널을 자동으로 잡아냅니다."

**자막:**
- 채널 그룹 관리 & 실시간 모니터링
- 채널 비교 분석 차트
- 급등 채널 자동 감지

---

## [핵심 기능 2] 1:15 ~ 1:40 — 소재 발굴

**화면:** 키워드 소재 탐색 시연 — 키워드 입력 → 결과 목록 → 필터 적용

**나레이션:**
"키워드 하나만 입력하면 지금 뜨고 있는 영상 소재를 바로 찾아드립니다.
24시간, 7일, 30일 단위로 필터링하고, 숏츠와 롱폼도 구분해서 탐색할 수 있어요.
마음에 드는 채널은 바로 내 모니터링 리스트에 일괄 저장하세요."

**자막:**
- 키워드 기반 소재 탐색
- 기간별 / 영상 유형별 필터
- 발견한 채널 일괄 저장

---

## [핵심 기능 3] 1:40 ~ 2:05 — 트렌드 분석

**화면:** 실시간 국가 트렌드 → 카테고리 트렌드 → 자동 탐색 Shorts 시연

**나레이션:**
"실시간 국가 트렌드로 지금 한국에서 가장 뜨는 영상을 확인하고,
카테고리별 트렌드로 내 분야의 인기 콘텐츠를 빠르게 파악하세요.
숏츠 자동 탐색은 바이럴 숏츠 채널을 알아서 찾아줍니다."

**자막:**
- 실시간 국가 & 카테고리 트렌드
- 숏츠 바이럴 채널 자동 발견

---

## [핵심 기능 4] 2:05 ~ 2:30 — AI 분석 & 골드 기능

**화면:** AI 바이럴 분석 결과 → 유사 썸네일 찾기 → 대본 추출 시연

**나레이션:**
"왜 이 영상이 터졌을까? AI가 바이럴 이유를 분석해드립니다.
골드 회원이라면 더 강력한 기능을 쓸 수 있어요.
AI 유사 썸네일 찾기로 비슷한 컨셉의 영상을 발굴하고,
대본 추출로 인기 영상의 스크립트를 바로 확인하세요."

**자막:**
- AI 바이럴 분석 (왜 이 영상이 떴을까?)
- AI 유사 썸네일 찾기
- 영상 대본(자막) 추출

**자막:**
- 전문가 큐레이션 추천 소재

# 촬영/편집 가이드

## 화면 구성
- 실제 서비스 화면 녹화 (크롬 전체화면 권장)
- 각 기능 시연 시 마우스 하이라이트 효과 추천
- 화면 전환은 슬라이드 또는 줌 트랜지션

## 나레이션
- 톤: 신뢰감 있고 밝은 목소리
- 속도: 보통~약간 빠르게 (정보 전달 위주)
- AI TTS 사용 시 추천: TypeCast, 클로바더빙

## BGM
- 밝고 경쾌한 테크 느낌
- 추천: Epidemic Sound "Corporate Tech" 카테고리

## 자막 스타일
- 핵심 키워드는 강조 색상 (노란색 or 브랜드 컬러)
- 하단 1/3 위치에 2줄 이내
- 기능 소개 시 좌측에 아이콘 + 텍스트 리스트

## 썸네일 제안
- 텍스트: "유튜브 소재, 아직도 수작업?" 또는 "AI가 찾아주는 유튜브 소재"
- 배경: 튜브레이더 대시보드 스크린샷 (블러 처리)
- 강조: 빨간색/노란색 텍스트 + 놀란 표정 인물 컷
